== Description ==

Minimal Business is a clean, responsive theme that’s versatile and easy to use. Suitable for business, corporate,  and creative agency websites. this  Themes design gives you maximum flexibility. This theme has full customization options along with full color and layout control from the customizer. Overall, if you’re looking for an intuitive, flexible, and lightweight WordPress theme Minimal Business free is a best for you. 

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
